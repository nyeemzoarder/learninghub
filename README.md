# Learning Hub

A collection of self-paced, modular Microsoft certification courses. Each
course pairs prerequisite/concept docs, draw.io architecture diagrams, and
hands-on labs.

## Courses

| Course | Status |
|--------|--------|
| [AZ-104: Microsoft Azure Administrator](courses/az-104/README.md) | Active |

To add a new course (e.g., AZ-204, AZ-305), create `courses/<course-id>/`
following the same structure as `courses/az-104/` — a `00-prerequisites/`
module, numbered topic modules (each with `README.md`, `concepts/`,
`diagrams/`, `labs/`), and a `resources/` folder. See
[_templates/](_templates/) for the per-module file templates.

## Shared resources

- [_templates/](_templates/) — module README, concept doc, and draw.io
  naming conventions reused across all courses
