Module 01 – Identity & Governance: Hands-On Labs

Practical exercises to master Entra ID, RBAC, and access control
📚 3 Labs ⏱ ~2.5 Hours Total 🎯 Intermediate
Before You Start

Make sure you've read the concept documents (01-05) for this module. Each lab builds on the foundational knowledge, and they're designed to be completed in order. You'll need an Azure subscription with Global Administrator or User Administrator role to complete these labs.

Labs Overview

These hands-on labs give you practical experience with the identity and governance concepts from the module. You'll work directly in Azure to create real-world scenarios, delegate permissions, and implement access controls.

Lab 01

Entra ID Users & Groups

⏱ 50 min 🟡 Intermediate

Create users and groups in Microsoft Entra ID, then use Administrative Units to delegate management permissions to department heads. Learn the critical difference between groups (WHO) and admin units (WHERE).

Start Lab →
Lab 02

RBAC & Azure Policy

⏱ 60 min 🟡 Intermediate

Assign role-based access control roles to users and groups at different scopes. Implement Azure Policy to enforce organizational standards and prevent misconfigurations at scale.

Start Lab →
Lab 03

Management Groups & Subscriptions

⏱ 40 min 🟡 Intermediate

Organize multiple subscriptions with management groups and apply policies at scale. Practice applying role assignments and policies across your entire organizational hierarchy.
