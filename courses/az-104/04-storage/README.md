## Learning objectives

By the end of this module you should be able to:
- Create storage accounts with appropriate redundancy and performance tiers
- Secure storage with access keys, SAS tokens, and network rules
- Work with blob containers, queues, tables, and file shares via the Storage
  browser
- Configure blob lifecycle management, versioning, point-in-time restore, and
  static website hosting
- Create Azure file shares, configure permissions, and set up Azure File Sync
